:root
Define variáveis globais. 
:root {
  --cor-principal: #2C3E50;
  --texto-claro: white;
}
Uso com var()
Utilização das variáveis no CSS. 
body {
  background-color: var(--cor-principal);
  color: var(--texto-claro);
}
Exemplo Prático
Este bloco usa as cores da variável: cor-principal e texto-claro.
