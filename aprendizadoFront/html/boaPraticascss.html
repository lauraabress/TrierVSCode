Aprendizado Front

Nomes de classes
Use letras minúsculas e kebab-case (palavras com hífen).
Ex: .menu-principal, .btn-verde
IDs
Use apenas para identificadores únicos. Evite usar IDs para estilizar, prefira classes.
Ex: #formPrincipal, #btnEnviar
Organização do CSS
Agrupe estilos por componentes ou blocos.
Ex: .card, .card-header, .btn, .btn-primary
Evite usar !important
Use somente em último caso. Dificulta manutenção e sobrescrita de estilos.
Separar CSS, HTML e JS
Prefira usar arquivos .css externos para facilitar a manutenção e organização do código.
Comentários claros
Utilize comentários para explicar blocos de código complexos.
Ex: /* Estilo do botão principal */
Consistência
Mantenha indentação padronizada (2 ou 4 espaços).
Escolha um estilo e siga-o por todo o projeto.
Acessibilidade
Use alt descritivo em imagens e garanta bom contraste de texto para facilitar leitura.
