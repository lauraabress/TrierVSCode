transition
Cria uma transição suave entre estados.
transition: all 0.3s ease;
Passe o mouse no quadrado azul.
transform
Aplica efeitos como rotate, scale, etc.
transform: scale(1.1);
(Já aplicado no exemplo acima.)
animation
Animações com keyframes.
animation: mover 2s infinite;
@keyframes
Define os passos de uma animação.
@keyframes mover {
  0% { transform: translateX(0); }
  100% { transform: translateX(100px); }
}
