🔁 let
Declara uma variável cujo valor pode mudar.
let contador = 0;
contador = contador + 1;
🔒 const
Declara um valor constante (não pode ser reatribuído).
const PI = 3.14;
⚠️ var
Forma antiga de declarar variáveis, evite usar. Tem escopo mais amplo e imprevisível.
var nome = "Ana";
