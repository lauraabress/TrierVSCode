Aplicam estilos aos elementos HTML que correspondem ao seletor especificado.
Elemento
Aplica a todos os elementos da <tag> especificada.
Exemplo: p { color: red; }

.classe
Aplica a todos os elementos com determinada class.
Exemplo:
.menu { background: gray; }
