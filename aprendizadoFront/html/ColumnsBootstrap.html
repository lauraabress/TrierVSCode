📝 Resumo:

Sistema de 12 colunas: O Bootstrap divide cada linha em 12 colunas imaginárias.

col-X: Onde X é o número de colunas que o elemento ocupará (1 a 12).

Breakpoints: sm (≥576px), md (≥768px), lg (≥992px), xl (≥1200px), xxl (≥1400px).

Responsividade: Use col-breakpoint-size para diferentes tamanhos de tela.
