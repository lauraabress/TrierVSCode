flex-wrap
Controla se os itens devem ou não quebrar para outra linha.

Exemplo: 

.container {
  display: flex;
  flex-wrap: wrap;
}

flex-wrap: nowrap
Os itens ficam em uma única linha, mesmo que transbordem. 

.container {
  display: flex;
  flex-wrap: nowrap;
}
1
2
3
4
5
flex-wrap: wrap-reverse
Quebra os itens em várias linhas, mas de baixo para cima. 

.container {
  display: flex;
  flex-wrap: wrap-reverse;
}

row-gap / column-gap
Controla o espaçamento entre linhas e colunas em um container flex ou grid. 

.container {
  display: flex;
  flex-wrap: wrap;
  row-gap: 10px;
  column-gap: 20px;
}
