📝 Nomes claros
Use nomes que expressem bem o propósito.
// Ruim
let x = 10;

// Bom
let idadeUsuario = 10;
📏 Funções pequenas
Funções devem fazer apenas uma coisa.
function calcularDesconto(preco, porcentagem) {
  return preco - (preco * porcentagem / 100);
}
📂 Código separado
Mantenha o JavaScript em arquivos externos:
<script src="scripts/main.js"></script>
💬 Comentários úteis
Comente trechos importantes ou complexos.
// Calcula o valor final com desconto aplicado
const precoFinal = calcularDesconto(100, 20);
🔒 const por padrão
Use const por padrão, só let se o valor mudar.
const nome = "João";
let idade = 25; // pode mudar
🌍 Evite variáveis globais
Use escopo local para evitar conflitos.
function saudacao() {
  let nome = "Maria";
  console.log("Olá " + nome);
}
