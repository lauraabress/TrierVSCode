Aprendizado Front

📝 string
Representa texto (entre aspas).
let mensagem = "Olá, mundo!";
🔢 number
Representa números, inteiros ou decimais.
let idade = 42;
let pi = 3.14;
✅ boolean
Representa verdadeiro ou falso.
let ativo = true;
let concluido = false;
🚫 null
Representa ausência proposital de valor.
let usuario = null;
❓ undefined
Variável declarada mas ainda sem valor.
let resultado;
📦 object
Conjunto de dados com chaves e valores.
let pessoa = {
  nome: "Maria",
  idade: 30
}
📚 array
Lista de valores em ordem.
let numeros = [1, 2, 3, 4];
