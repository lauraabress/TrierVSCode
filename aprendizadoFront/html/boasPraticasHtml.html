Aprendizado Front

Usar tags semanticas
Prefira (main), (section), (article), (nav), (header), (footer), (aside) para estruturar o conteúdo com significado.
Manter o documento organizado
Use indentação consistente (2 ou 4 espaços) para facilitar leitura.
Usar atributos alt em imagens
Sempre coloque alt descritivo nas <img> para acessibilidade.
Usar labels associados a inputs
Relacione <label> com <input> usando o atributo for para melhorar usabilidade e acessibilidade.
Evitar tags obsoletas
Não usar tags antigas como <font>, <center>, prefira CSS para estilo.
Fechar todas as tags corretamente
Para evitar erros no navegador.
Separar estrutura e estilo
HTML para conteúdo e estrutura, CSS para aparência.
Evitar inline styles
Use classes e arquivos CSS externos.
Validar o código HTML
Utilize ferramentas como o W3C Validator para garantir qualidade.
