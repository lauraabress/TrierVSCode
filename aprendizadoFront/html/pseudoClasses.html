:hover
Quando o mouse está sobre o elemento.
a:hover { color: red; }
Passe o mouse aqui
:active
Enquanto o botão está sendo clicado.
button:active { transform: scale(0.95); }
:focus
Quando o elemento está em foco.
input:focus { border-color: blue; }
:nth-child(n)
Seleciona o n-ésimo filho de um elemento.
li:nth-child(2) { color: green; }
